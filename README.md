# Bike-Sales-Analysis-Dashboard
               Bike Sales Data Analysis Project
                                 By Shivam

This project analyses customer data to understand factors influencing bike purchases using Excel and this process involved stages, including data pre-processing, data cleaning and visualization. As result:- Interactive dashboard was created to visualize patterns in sales.

## Objectives :-

Analyse customer behaviour - Identify key sales factors - Build interactive dashboard - Extract insights

## Tools & Techniques :-

Microsoft Excel, Data Cleaning, Pivot Tables, Pivot Charts, Slicers, Data Transformation


## Data Set Used :-

<a href="https://github.com/Shivam62563/Bike-Sales-Analysis-Dashboard/blob/main/Excel%20Project%20Dataset.xlsx"> Bike Sales Data</a>


## Work Done :-

Cleaned dataset, created Age Bracket column, built pivot tables and charts, designed dashboard,
added slicers


## Key Insights :-

Short commute customers buy more bikes; Middle-aged customers dominate purchases;
Moderate/high income customers buy more; Professionals more likely to purchase


## Challenges :-

Handling large data, designing dashboard, selecting correct charts
